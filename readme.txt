「聖邪の天使」強化MOD 0.0.0。
 - 全体の変更
   - 耐久値アップ
   - 攻撃の組み合わせが増える（128種類）
   - 隙を短縮
 - 攻撃別の変更
   - 炎：発射頻度が上がる
   - 氷：攻撃回数の増加
   - 風：ダッシュ速度、回数アップ
   - 地：発射間隔が短い
   - 雷：攻撃回数の増加
   - 光：変化なし

使用方法
   1. xdelta3のダウンロード
   2. z64フォーマット（ビッグエンディアン）ROM
   3. xdelta3 -d -q -s [ROMファイル] baku2.xdelta3 [出力ファイル]。
